### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/ClawGibson/TodoApp.git
   ```
2. Install dependencies
   ```sh
   yarn
   ```
3. Create your .env file `root\.env`
   ```.env
   MONGODB = 'Your mongodb url connection'
   API_URL = 'Your api url'
   DBNAME = 'Your database name'
   SECRET = 'Your secret'
   ```
4. Run
   ```sh
   yarn dev
    
   yarn start
   ```
